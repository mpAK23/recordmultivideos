OverdubVideo

Instructions

Equipment Required

  • Desktop or Tablet with a Web Browser (Chrome recommended)
  • Phone with browser which serves as a camera
  • Headphones for monitoring if recording more than one video

Overdub Video allows for recording up to four videos with tools that aid in having them in sync.

On a desktop computer or tablet click the "Create Control Center" button. Upon entry, at the top you will see a code. Open the same website on a phone and enter the code in the block on the page labeled "Camera Mode." From the control center click the + for the first block you want to record. You will see the view your phone has. It uses the rear camera.

Once ready to record, click the red record button. It will count down. Click to stop. You can play it back. You can delete it. When ready to add a second track, you can see the tick in the first track "Mon." Make sure that is ticked and with headsets listen to the first track as you play along. Again, it will count down.

Once the tracks are recorded there is also an Exp tick where you can decide if you want all recorded videos to be included in the export.

Finally, before hitting export, you can name the file.
